# Create New Research Prompt

You are helping the user create a new research prompt for their ADHD medication access research project.

## Task

1. Ask the user for:
   - The research question or topic
   - Whether this is a draft (goes to prompts/drafting) or ready to run (goes to prompts/to-run)
   - Any specific areas of focus or jurisdictions to consider

2. Create a well-structured prompt file with:
   - Clear research question as title
   - Context from the context/ folder that's relevant
   - Specific deliverables expected
   - Any data sources to consult
   - Timestamp and metadata

3. Save the prompt file with a descriptive filename in the appropriate folder:
   - Format: `YYYY-MM-DD-topic-slug.md`
   - Example: `2025-11-07-uk-adhd-prescription-limits.md`

4. Confirm creation and show the file path

## Guidelines

- Make prompts specific and actionable
- Reference relevant context files
- Include expected output format
- Consider comparative analysis opportunities
- Note any data extraction needs
